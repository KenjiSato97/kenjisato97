# Olá, eu sou Kenji Sato! 👋

## 👨‍💻 Sobre Mim

Sou um **Desenvolvedor focado em Dados**, apaixonado por transformar informações brutas em insights acionáveis e interfaces funcionais. Minha jornada é movida pela curiosidade de entender como sistemas complexos funcionam e como construir a ponte entre o banco de dados e a experiência do usuário.

* **Impacto Educacional:** Desenvolvi uma plataforma que utiliza imagens aeroespaciais para o ensino de geografia e uma plataforma para avaliação de competencias e habilidades.
* **Finanças & Gestão:** Criei um ecossistema de gestão financeira pessoal com módulos de automação e controle bancário.
* **Análise Estratégica:** Realizo projetos complexos de análise, como o cruzamento de dados de e-commerce (**Olist**) com dados demográficos oficiais (**IBGE**).
